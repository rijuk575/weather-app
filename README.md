Weather App

A responsive weather application that provides real-time data, location-based forecasts, and interactive visualizations.

## Features

* Real-time weather by city search
* Automatic location detection (Geolocation API)
* 5-day forecast with daily summaries
* Temperature trend visualization using Chart.js
* Dark mode with persistent user preference
* Clean, modern glassmorphism UI

##  Tech Stack

* HTML, CSS, JavaScript
* Chart.js
* OpenWeather API

##  Setup

```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app
```

Add your API key in `script.js`:

```js
const apiKey = "API_KEY_HERE";
```

Open `index.html` in your browser.
